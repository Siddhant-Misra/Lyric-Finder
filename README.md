# LyricFinder App
> A REACTJS App that finds song lyrics.

## Musixmatch API
Get your API key and add to the .ENV file
[https://developer.musixmatch.com](https://developer.musixmatch.com)

## Quick Start
1.Install NodeJS
npm install

2.To run on localhost
npm start

3.Building for production
npm run build
